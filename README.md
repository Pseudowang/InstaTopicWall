# InstaTopicWall

InstaTopicWall是一个Rails应用程序，允许用户创建主题并从Instagram获取相关帖子，用于展示和分享.


### 本地使用

1. 克隆仓库
```bash
git clone https://github.com/yourusername/InstaTopicWall.git
cd InstaTopicWall
```

2. 安装依赖
```bash
bundle install
```

3. 设置数据库
```bash
rails db:create
rails db:migrate
```

4. 启动服务器

```bash

rails s -b 0.0.0.0 # 仅启动Rails服务器
```

5. 访问应用
打开浏览器访问 `http://localhost:3000`
